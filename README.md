doctus
======

Multiplatform GUI-Framework

For details see the [doctus homepage](http://entelijan.net/art/doctus/index.html) 

## run the showcases

In the project base directory call
```shell
sbt 
```
Inside sbt call
```shell
publishLocal 
```
To call a 'JVM' showcase call
```shell
run
```
Select one of the mainclasses
